简谱数据格式设计与可视化

本演示展示了一个完整的简谱数据格式设计，包含音符、时值、调号、拍号等核心元素，并提供了JSON编辑与实时可视化功能

JSON 编辑器

简谱可视化

数据结构说明

metadata 对象
title
乐曲标题
composer
作曲者
key
调号 (如"1=C")
timeSignature
拍号 (如"4/4")
tempo
速度 (BPM)
measures 数组
index
小节序号
notes
音符数组
note 对象
type
"note" 或 "rest"
pitch
音高 (1-7，休止符为0)
octaveShift
音区偏移 (0=中音, -1=低音, 1=高音)
duration
基础时值 (1=四分音符)
dotted
是否附点音符
lyric
歌词 (可选)

使用说明

  • 在左侧编辑JSON数据后，点击"渲染简谱"按钮更新可视化
  • 点击"小星星"、"欢乐颂"或"C大调进阶"按钮加载示例
  • 悬停在音符上可查看详细信息
  • 修改数据后需要重新渲染以更新视图
  • 点击"画布窗口"按钮可在独立窗口中查看Canvas渲染效果
